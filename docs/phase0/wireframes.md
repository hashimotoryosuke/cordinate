---

## デザイン方針

- **シンプル・ミニマル**: ファッション初心者が迷わない UI
- **大きな画像**: 服・コーデは視覚が重要。画像を大きく見せる
- **1アクション1画面**: 迷わないよう、1画面で求めるアクションは1つ
- **ポジティブなコピー**: 「あなたに似合う」「おしゃれに見える」など前向きな表現

## デザイントークン（実装済み）

`packages/ui/src/tokens.ts` に定義済み。

| トークン | 値 | 用途 |
|---------|-----|------|
| `colors.primary` | `#1A1A1A` | メインボタン・テキスト |
| `colors.accent` | `#C9A84C` | アクセント（ゴールド） |
| `colors.background` | `#FAFAFA` | 背景 |
| `colors.muted` | `#F4F4F5` | カード背景 |
| `typography.fontFamily.serif` | Playfair Display | 見出し |
| `typography.fontFamily.sans` | Inter | 本文 |

## コンポーネントライブラリ

- **Web**: shadcn/ui（Radix UI ベース、Tailwind CSS v4 対応）
- **Mobile**: React Native の標準コンポーネント + NativeWind（Tailwind を RN に適用）
